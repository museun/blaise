# blaise
some pascal thing


````cargo run -- some_standard_pascal.pas````

some useful environmental variables:

| $env | what it prints |
| --- | --- |
| `BLAISE_SOURCE` | the input source |
| `BLAISE_TOKENS` | the token stream |
| `BLAISE_AST` | the AST |
| `BLAISE_TRACE` | a tracer for the parser |
